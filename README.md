# Alireza Nobakht Portfolio

A fully static, GitHub Pages-ready personal portfolio for Alireza Nobakht.

## Deploy

1. Create a GitHub repository, for example `alireza-nobakht-portfolio`.
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Set **Source** to `Deploy from a branch`.
5. Choose `main` and `/root`.
6. Save. The site will be available from GitHub Pages after the deployment finishes.

No backend, build step, database, paid API, or framework is required.


Logo update: the uploaded AN logo is included as `assets/icons/an-logo.png` and used for the site favicon and logo placeholders.
